# TabMerge

Windows 文件资源管理器窗口标签合并工具 —— 新打开的资源管理器窗口自动合并到已有窗口的标签页中，类似浏览器标签合并。

## 适配系统

- Windows 11 22H2 及以上版本
- 系统文件资源管理器需支持原生标签页功能
- 需要安装 .NET Framework 4.8

> 说明：本工具依赖 Windows 11 文件资源管理器的原生标签页能力，通过 UI Automation 和键盘模拟完成窗口合并。Windows 10 或不支持资源管理器标签页的 Windows 11 版本不适用。
>
> 判断方式：打开文件资源管理器后，按 `Ctrl + T`。如果能在当前资源管理器窗口中新建一个标签页，说明当前系统支持；如果没有反应或打开了新窗口，则当前系统不适用。也可以查看资源管理器窗口顶部是否有类似浏览器的标签栏。

## 单文件系统
只有150KB，运行后占用内存大概25MB左右
## 功能

- 新资源管理器窗口自动合并为标签页
- 系统托盘运行，后台静默工作
- 支持开机启动
- 一键启用/关闭合并

## 技术栈

- .NET Framework 4.8 WinForms
- SHDocVw COM 互操作
- UI Automation
